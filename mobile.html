This is a portfolio based on Discord's app design, which I was motivated to make after seeing a few other projects designed around other desktop apps. I thought it would be a fun challenge to take something I use regularly and recreate it.

Part of 100 Days of Code

I'm currently challenging myself to code for at least an hour a day for a minimum of 100 days. This project is day two of the challenge. You can see some of my other projects in the "Other Projects" part of this pen.

Project Breakdown

I gave myself 3 hours to build this, and found it easy to make something that resembles the original product. However, there are still some things I'd like to include if I revisit this at a later date such as:

  • Member list: I didn't really know what I could use this for in terms of a portfolio context, but if I think of a use, I'd definitely like to implement this.
  • Mobile layout: Since the mobile layout is so different to the desktop, this would have been a whole other project. Also involving more complex JavaScript.
  • Images: I'd have liked to include some appropriate graphics to make the design a little more interesting, but my main reason for doing this is to improve my code skills rather than image sourcing. If I come across some graphics that would fit well, I'll likely add these at a later date.
  • Buttons: It would have been quick to add styled buttons, but this is another thing where I wasn't sure where to use them.

I was also limited by certain things such as only using web fonts, and using FontAwesome instead of custom icons, but I did my best to get as close to the look as I could.
